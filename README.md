  <b>ARC</b>
</h1>

<p>
  The ARC (Augmented Retrieval Chatbot) is an intelligent, document-grounded conversational assistant designed to mitigate the inefficiencies of manual information extraction from large files. By leveraging Retrieval-Augmented Generation (RAG), ARC allows users to upload various document formats and query their contents using natural language. The system provides highly accurate, context-aware responses, reducing hallucinations typical of standard Large Language Models (LLMs).
</p>
